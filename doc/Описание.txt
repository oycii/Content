       *  Для мелодий все возможные форматы в одном экспорте.
       *  Значительно меньше размер выгрузки

 Особенности работы.

 Код для получения контента.
   Для  каждого  формата  существует  своя маска для кода заказа.
   Пример:! Видео
       <media_format id="27" name="Видео" code_mask="27xxxx9999"
   Маска кода заказа 27xxxx9999, теперь если id контента равно 321
   то  код  заказа  будет  2703219999.  Вместо  xxxx  подставляется id
   контента,  выравненное  по правой стороне. Места остающиеся пустыми
 заполняются нулями.

  Сервисный номер для получения заказа.
  У  каждой  единицы  контента  item  есть  аттрибут   fi_ids - format
  instance.  Группа  связывающая  формат контента со списком сервисных
  номеров для получения его.
    <format_instance id="227" mf_id="27" sg_id="200"/>
    27 формат  - это видео
    200 сервисная группа.
  Список   операторов   и  их  сервисных  номеров  находится  в  теге
  <service_group id="200" price="2.00">

 
В приложенных файлах высылаем Вам dtd и небольшое описание
